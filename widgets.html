DONATION BUTTONS

Donate

Requires the
btn-donate class

Balance: 0.00000000 | Donations: 0

The only required options here are address and chain - which should be added to the HTML data-attribute as follows:

<a href="#" data-address="naj1CrvX2kBXa7uDJMRfiUAnWD5ZFvGS54" data-chain="doget">Donate</a>
Other HTML Data-Attribute Options:

  • data-qr default is set to true

    Displaying QR Codes requires the qrcode.js dependency.

  • data-bip default is set to true

    Would you like to display Desktop QT Link?

  • data-amount default is set to 0

    Include defined amount in BIP Link and Modal Header?

  • data-inject default is set to true

    Attempt to auto-inject modal HTML?

  • data-group default is set to false

    If updating text we use this as the class and add the dynamic text to an element with the following class, as follows (and seen above):

    <span class="dontate-txs-naj1CrvX2kBXa7uDJMRfiUAnWD5ZFvGS54">0</span>

  • data-balance default is set to false

    Update balances based on group class and address.

  • data-txs default is set to false

    Update TX counts based on group class and address.

  • data-loading default is set to false

    This is the text to display prior to fetching dynamic content and will use the content within the HTML if nothing specific has been set.

  • data-label default is set to false

    This allows you to add a label to Desktop QT links.
